# Retail Sales Analytics
Proyecto de análisis de desempeño comercial de una empresa retail utilizando Tableu Public y SQL.

Objetivo: Analizar el desempeño comercial de una empresa retail para identificar oportunidades de crecimiento y rentabilidad.

## Tecnologías
- SQL Server
- Tableau Public

## Hallazgos
- Technology es la categoría más rentable.
- West concentra el mayor volumen de ventas.
- Existe una fuerte concentración de ingresos en pocos clientes.
- Algunas subcategorías generan pérdidas pese a registrar ventas.
